================================================================================
                         QUICK START
================================================================================

1. Clone repository
   git clone https://github.com/RishavOsaurus/p2p-file-sharing.git

2. Install dependencies
   cd server && npm install
   cd ../client && npm install

3. Run tests
   cd server && npm test
   cd ../client && npm test

4. Start servers
   Terminal 1: cd server && npm run dev
   Terminal 2: cd client && npm run dev

5. Open browser
   http://localhost:3000

6. Test file transfer
   - Join same room on multiple tabs/devices
   - Select files and send
   - Accept on receiver
   - Watch progress and download
